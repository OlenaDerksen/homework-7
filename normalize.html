Normalize.css is a small CSS library that we use to ensure greater consistency in the display of web pages across different browsers. By including the file, we can see a different font from the index.html file. The bold font of the STRONG tag is preserved. An inline element is displayed at the bottom relative to the block element next to it. Also, Normalize.css applies the box-sizing property, which ensures that paddings and borders do not increase the size of the block element.
